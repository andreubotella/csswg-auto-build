CSS Styling Attributes Disposition of Comments

Last call document:http://www.w3.org/TR/2010/WD-css-style-attr-20100121/

Editor's draft: http://dev.w3.org/csswg/css-style-attr/

The following color coding convention is used for comments:

  • Accepted or Rejected/OutOfScope/Invalid + verified
  • Rejected and not verified
  • Out-of-Scope or Invalid and not verified
  • Formal Objection (There were no formal objections.)

An issue can be closed as Accepted, OutOfScope, Invalid, Rejected, or Retracted. Verified indicates commentor's acceptance of the response.
